                        ***************  CLI приложение-парсер (PHP)  ***************

Для запуска парсера, у вас на ПК должен быть установлен PHP Интерпретатор!

---------------------------------------------
ИНСТРУКЦИЯ:

- 1. Откройте папку парсера в теминале.
- 2. Для запуска программы, команда: php start
- 3. Введите желаемую команду.

    3.1 Список команд:
        parse - Запускает парсер. Принимает обязательный параметр URL (как с протоколом, так и без).
        report - Выводит в консоль результаты анализа для домена. Принимает обязательный параметр Domain (как с протоколом, так и без).
        help - Выводит список команд с пояснениями.
        

- 4. Отчеты находятся в папке "report_files".

    4.1 Список сылок с указаного адреса - "домен"_List_Links.csv
    4.2 Список изображений с указаного адреса - "домен"_List_Image.csv
